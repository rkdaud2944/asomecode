SystemMessage

"Test" 버튼을 클릭하면 화면에 메시지가 표시됩니다. 이 메시지는 어플리케이션 영역에서 처리됩니다.


InstallDriver

"Test" 버튼을 클릭하면 드라이버 설치 프로그램이 실행됩니다.


호스트 장비의 메모리 사용

# 호스트 메모리에 문자열 저장 print("### System.Line.MainJS: App.SetString('id', 'test-value')")
실행하기
멈추기

"실행하기" 버튼을 클릭하면 파이썬 보드에서 파이썬 코드가 실행되어 호스트 메모리에 문자열이 저장됩니다.

저장된 이후에는 아래와 같이 웹에서 호스트 메모리에 접근하여 사용할 수 있습니다. 각종 시뮬레이션 등에서 호스트와 메모리를 공유하여 통신할 때 사용됩니다. "자바스크립트로 확인하기" 버튼을 클릭하면 아래 이미지와 같이 저장된 문자열을 확인할 수 있습니다.

"자바스크립트로 확인하기" 버튼에 사용된 자바스크립트 코드는 아래와 같습니다. 

alert(App.GetString('id'));

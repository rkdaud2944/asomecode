어썸블록안드로이드(WiFi) , 어썸 보드 연결하기

1. 어썸블록안드로이드 (WiFi) 설치 및 실행


  1. 어썸블록안드로이드(WiFi)버전을 구글 Playstore 에서 다운로드 받습니다.

  2. 다운로드 받은 어썸블록안드로이드 파일을 실행시킵니다. 이때 핸드폰/테블릿 PC는 인터넷에 연결되어 있어야 합니다.(어썸보드 WiFi 연결 X)

  3. 원하는 국가와 언어의 버전을 선택합니다.

  4. 화면 오른쪽 상단의 버튼을 클릭하여 Block 버튼을 클릭하여 블록코딩을 준비합니다.

2. 어썸보드 설정하기


  1. 어썸코드(블록)에서 어썸보드을 연결 후 v1.12이 맞는지 확인합니다. 다른 버전일 경우 아래 [Prepare AsomeBoard]를 먼저 진행하여 주세요.

  2. v1.12 버전의 어썸보드에 텔넷 서버를 업로드 합니다. 아래의 [Upload telnet server]를 따라 진행하여 주세요.

  3. 업로드가 완료되면 연결된 USB 케이블을 해제하고, 어썸보드 전원을 한번 껐다 켜줍니다.

3. 어썸봇안드로이드 어플과 어썸보드 연결하기


  1. 어썸봇 안드로이드 어플이 실행 되어있는 상태에서 아까 설정한 WiFi를 검색 후 연결합니다.

  2. 어플로 돌아와서 블록코딩 창의 connect 버튼을 클릭합니다.

  3. 기존의 블록 코딩하기와 같은 방식으로 코딩을 진행하여 어썸봇(카)를 움직입니다.

  4. 어썸블록안드로이드(WiFi) 사용이 끝나거나, 엔트리를 사용하여 코딩하고 싶다면 어썸봇을 다시 연결한 후 페이지 가장 아래의 [자동 실행 프로그램 삭제]를 한 후 사용하셔야 합니다.




유의 사항

인터넷/ UDP 소켓 일부 예제 관련

  1. 어썸블록안드로이드(WiFi)버전은 AP모드(공유기)입니다.
  2. UDP 소켓 일부 에제를 제외한 인터넷 관련 교재내용의 동작이 정상 작동 하지 않을 수 있습니다.
  3. 블록 코딩 과정에서도 일부 인터넷사용 블록에 제한이 있을 수 있습니다.




Prepare AsomeBoard

USB 케이블을 사용하지 않고 WiFi를 이용하여 어썸보드에 연결하려면 v1.12 버전의 펌웨어를 사용해야 합니다.
아래 펌웨어를 다운받고, 아래 링크의 [AsomeBoard-Zet 공장 초기화]를 순서대로 진행해 v1.12 버전의 보드를 업로드 해주세요


펌웨어 다운로드 : 어썸보드 1.12버전 펌웨어 다운로드

공장초기화 하는 법 : [어썸보드 v1.12 버전 업데이트 하러가기]




Upload telnet server

원격으로 명령어를 처리할 수 있는 텔넷 서버를 어썸보드에 올리기 위해서 WiFi(SSID) 이름과 암호를 입력하시고
[Telnet server 등록] 버튼을 클릭하세요.

  • 아이디와 암호는 영문과 숫자 조합을 사용해야 합니다.
  • 암호는 8자리 이상이어야 합니다.
  • 어썸보드는 어썸코드에 연결되어 있어야 합니다.

code = """ import internet import telnet_server internet.open_ap("{0}", "{1}") telnet_server.start() """ f = open("main.py", "w") f.write(code) f.close()

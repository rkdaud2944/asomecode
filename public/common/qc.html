가. 중심잡기

어썸봇이 똑바로 걷거나 이동하기 위해서는 중심이 정확하게 잡혀있어야 합니다. 하지만 모터 안의 기어가 공장에서 나올 때 조금씩 다를 수 있기 때문에 각 모터의 중심 각도를 찾아서 asomebot 모듈에 적용하시면 보다 정확하게 걷게 됩니다.

어썸봇을 컴퓨터에 연결하고 [연결하기] 버튼을 클릭하세요. 그리고 아래 버튼의 숫자를 조정하며 정확한 차렷 자세를 만들어 보세요 [▲] [▼] 버튼을 클릭하거나 원하는 숫자를 입력하여 [차렷자세 테스트 하기] 버튼을 클릭하면 어썸봇 모터들이 중심으로 이동합니다. 조금 삐뚤어져 있다면 숫자를 조금씩 변경해보세요.

어썸키트 테스트 (Pro 보드)

  • 어썸키트를 컴퓨터에 연결하고 [실행하기] 버튼을 클릭하세요.
  • Pro 버전의 어썸보드에서만 테스트가 가능합니다. (보드를 어썸코드와[연결하기]한 후 상단에 보면 Pro / Zet 확인 가능합니다.)
  • 테스트의 순서는 아래와 같습니다.
  • 초음파 센서 -> LED ,RGB -> 조도센서 -> 온습도센서 -> 로터리 인코더 -> 능동부저 -> 수동부저 -> 진동센서
  • 테스트를 원하지 않는 부분은 코드의 가장 앞에 # 을 붙이거나 드래그 한후 [ctrl+ /] 단축키를 이용하시면 주석 처리 하면 무시하고 코드 실행이 가능합니다.

import hcsr04 import tm1637 import dht11 import music import button import rotary import vibration_sensor hcsr04.open(7, 8) tm1637.open(3, 4) # 1. 초음파 센서 테스트 - tm1637 부품에 초음파센서의 측정값이 나타납니다 # 3cm 이하에 손을 가져두면 다음 단계로 넘어갈 수 있습니다. while True: tm1637.number(hcsr04.get_distance()) delay(0.5) if hcsr04.get_distance() != -1: if hcsr04.get_distance() < 3: break vibration = vibration_sensor.create(1, 1000) # LED 테스트 - 코드가 입력되면 모든 불이 켜집니다. pin = OutputPin(13) pin.on() pin = OutputPin(14) pin.on() pin = OutputPin(15) pin.on() pin = OutputPin(20) pin.on() # RGB - 코드가 입력되면 RGB 램프에 랜덤한 색이 켜집니다. red = ServoPin(17) green = ServoPin(18) blue = ServoPin(19) red.duty( random(1024) ) green.duty( random(1024) ) blue.duty( random(1024) ) encoder = rotary.create(9, 10) bt1 = button.create(6) bt2 = button.create(2) dht = dht11.create(5) dht.measure() # 2. 조도센서 - tm1637 에 조도센서 측정값을 보여줍니다. # 조도센서를 손으로 가리면 값이 변경되는것을 확인 할 수 있습니다. # 다음단계로 넘어가기 위해서는 버튼을 한번 클릭합니다. pin = AnalogPin(1) while True: tm1637.number(pin.read()) delay(0.3) if bt1.is_clicked(): break # 3. 온습도센서 - tm1637에 현재 온도값을 보여줍니다. # 다음단계로 넘어가기 위해서는 버튼을 한번 클릭합니다. while True: tm1637.number(dht.temperature()) delay(0.3) if bt1.is_clicked(): break # 4. 로터리인코더 - 로터리 인코더를 오른쪽, 왼쪽으로 돌리면 tm1637의 값이 변합니다. # 다음단계로 넘어가기 위해서는 로터리 인코더를 클릭합니다. while True: encoder.update() tm1637.number(encoder.value()) if bt2.is_clicked(): break # 5. 능동부저 - 자동으로 소리가 울립니다. pin = OutputPin(11) pin.on() delay(1) pin.off() # 6 . 수동부저 - 자동으로 소리가 울립니다. music.open(12) music.note("C4", 300) music.close() # 7. 진동센서 - 진동센서를 테스트합니다. 어섬키트를 흔들면 다음단계로 넘어갈 수 있습니다. while True: if vibration.is_active(): break turnoff_pins() red = OutputPin(13) # 8. 소리센서 - 소리를 측정하여 소리가 들리면 빨간색 불을 켭니다. pin = AnalogPin(2) while True: tm1637.number(pin.read()) if pin.read() < 4000: red.on() delay(1) else: red.off()

전원 공급 테스트 (어썸킷 일부 부품)

아래 테스트는 전원 공급만으로 테스트 할 수 있는 일부 부품을 위한 프로그램입니다. 전원 공급 테스트로 불량을 알아 볼 수 있는 부품들은 아래와 같습니다.

  • LED 종류(초룩, 노랑, 빨강, 고휘도, RGB)
  • 부저 (수동부저, 능동부저)

양극과 음극에 상관없이 점퍼선 하나는 1번 디지털(노란색) 핀에 다른 하나는 2번 디지털 핀에 연결하기만 하면 됩니다. LED는 불이 들어오는 것을 확인하고 부저는 소리나는 것을 확인하면 됩니다.

RGB LED의 경우에는 RGB의 음극 점퍼선을 1번이나 2번에 연결하고 나머지 세 개 점퍼선을 다른 핀에 번갈아서 꽂아서 빨간색, 초록색, 파란색으로 변하는 지 확인하세요.

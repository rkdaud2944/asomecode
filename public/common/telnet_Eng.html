Connecting AsomeBlock Android (WiFi) and AsomeBoard

1. Installing and running the AsomeBlock Android (WiFi)


  1. Download the AsomeBlock Android (WiFi) version from Google Play Store.

  2. Run the downloaded AsomeBlock Android file. At this time, the mobile phone or tablet PC must be connected to the Internet (AsomeBoard WiFi connection X)

  3. Select the desired country and language version.

  4. Click the button at the top right of the screen, and then click the Block button to prepare the block coding.

2. Setting up the AsomeBoard


  1. After connecting the AsomeBoard in the AsomeCode (block), check whether it is v1.12. If it is another version, please run the ‘Prepare AsomeBoard’ at the bottom.

  2. Upload the Telnet Server on the v1.12 version of AsomeBoard. Please proceed by following the [Upload Telnet Server] part below.

  3. When the upload is completed, disconnect the connected USB cable and restart the AsomeBoard.

3. Connecting the AsomeBot Android application and the AsomeBoard


  1. With the AsomeBot Android application running, search for the previously set WiFi and connect.

  2. Return to the application and click the Connect button in the block coding window.

  3. Move the AsomeBot (car) by coding in the same way as the previous block coding.

  4. If you have completed using the AsomeBlock Android (WiFi), or if you want to code using an entry, you must reconnect the AsomeBot and use the [Delete Autorun Program] at the bottom of this page.




Notice

Some examples related to Internet and UDP sockets

  1. AsomeBlock Android (WiFi) version is AP mode (router).
  2. The Internet-related operation mentioned in textbooks may not work properly except for some examples of UDP sockets.
  3. In the block coding process, some restrictions may apply to the Internet use of blocks.

Upload telnet server

To upload a Telnet Server that can remotely process commands to the AsomeBoard, enter the WiFi (SSID) name and password,
click the [Telnet Server Registration] button

  • The ID and password must use a combination of alphabetical letters and numbers.
  • Password must be in at least 8 digits.
  • The AsomeBoard must be connected to the AsomeCode.

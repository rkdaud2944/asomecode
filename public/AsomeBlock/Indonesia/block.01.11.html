  • 3. Pelajari blok mana yang akan digunakan

    • A. Ubah ke mode titik akses/access point (grup [INTERNET])

    Beri nama AsomeBot Anda yang akan dikendalikan oleh ponsel cerdas.

    import internet internet.open_ap("")

    ① Masukkan nama yang ingin Anda gunakan. Hanya bahasa Inggris dan angka yang dapat digunakan.

    ① Beri nama AsomeBot. Hanya bahasa Inggris dan angka yang dapat digunakan untuk memberi nama AsomeBot. Setelah Anda menetapkan nama, nama tersebut akan tetap dipertahankan meskipun daya dihidupkan dan dimatikan.

    ② Periksa nama yang Anda tentukan dan coba sambungkan. Pertama, cari nama AsomeBot yang Anda masukkan dari daftar WiFi dan sambungkan ke ponsel cerdas Anda. Jika sebuah pesan muncul mengatakan bahwa koneksi Internet tidak stabil, jangan tekan tombol apa pun dan tunggu pesannya menghilang.


    B. Buka/Open UDP socket (grup [INTERNET])

    Buka port untuk AsomeBot untuk berkomunikasi melalui Internet.

    import udp_socket udp_socket.open(1234)

    ① Nomor port.Gunakan sebagaimana adanya tanpa modifikasi


    C. Baca Pesan (grup [INTERNET])

    Baca pesan dari smartphone.

    msg = udp_socket.read_text()

    D. Ulangi tanpa batas/repeat infinitely (grup [ADVANCE])

    [Repeat infinitely] membuat blok-blok yang didaftarkan ke dalamnya untuk terus berjalan berurutan tanpa henti. Kali ini, Blok ini digunakan oleh aplikasi telepon pintar untuk menunggu pesan dikirim, dan kemudian menjalankan perintah sesuai pesan saat diminta.

    while True: pass

    E. Jika/if (grup [ADVANCE])

    [If] membuat blok yang dimasukkan ke dalamnya untuk dijalankan kalau itu sesuai dengan kondisinya. Jika tidak sesuai dengan kondisi, maka blok di belakang blok [If] yang dijalankan.

    if i == :

    ① Masukkan variabel yang akan digunakan dalam pernyataan bersyarat. Menambahkan blok sensor ke nilai default ' i ' akan menambah variabel dan dapat dipilih.

    ② Pilih ketidaksetaraan default untuk membandingkan kondisi.

    ③ Masukkan nilai default untuk digunakan dalam kondisi ini. Anda perlu menambahkan blok untuk menentukan nilai bersyarat.

    ④ Tambahkan blok yang akan dieksekusi ketika kondisinya benar.

    G. Instal Asome Controller

    Aplikasi AsomeController harus diinstal untuk mengontrol AsomeBot secara nirkabel. Aplikasi ini memungkinkan Anda untuk mengirim pesan ke Asomebot dan mengontrol AsomeBot. Cari ' asomeit ' di Play store. Temukan dan unduh ' Asome Controller ' .


    Beri nama Asomebot Anda, buat ia tersedia/dapat diakses via Internet, dan buat program yang memenuhi persyaratan berikut.

    * Tunggu aplikasi ponsel cerdas untuk mengirim pesan.
    * Jika pesannya adalah " H " , berdirilah siaga [Attention].
    * Jika pesannya adalah " F " , maju/move forward [satu langkah ke depan].
    * Jika pesannya adalah " B " , mundur/move backward [satu langkah ke belakang].
    * Jika pesannya adalah " L " , Putar kiri/turn left [sekali].
    * Jika pesannya adalah " R " , Putar kanan/turn right [sekali].

    import internet internet.open_ap("asomebot01") import udp_socket udp_socket.open(1234) while True: msg = udp_socket.read_text() if msg == 'H': asomebot.home() if msg == 'F': for i in range(1): asomebot.forward() if msg == 'B': for i in range(1): asomebot.backward() if msg == 'L': for i in range(1): asomebot.turn_left() if msg == 'R': for i in range(1): asomebot.turn_right()

    G. Install Asome Controller

    The AsomeController app must be installed to control AsomeBot wirelessly. This app allows you to send messages to Asomebot and to control AsomeBot. Search ‘asomeit’ in the App store. Find and download ‘Asome Controller’.


    Name your Asomebot, make it available on the Internet, and create a program that meets the following conditions.

    * Wait for the smartphone app to send a message.
    * If the message is “H”, stand at [attention].
    * If the message is “F”, move [one step forward].
    * If the message is “B”, move [one step backward].
    * If the message is “L”, Turn [once to left].
    * If the message is “R”, Turn [once to right].

    import internet internet.open_ap("asomebot01") import udp_socket udp_socket.open(1234) while True: msg = udp_socket.read_text() if msg == 'H': asomebot.home() if msg == 'F': for i in range(1): asomebot.forward() if msg == 'B': for i in range(1): asomebot.backward() if msg == 'L': for i in range(1): asomebot.turn_left() if msg == 'R': for i in range(1): asomebot.turn_right()

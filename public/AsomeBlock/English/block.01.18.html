  • 2. Preview

    • A. How do you forecast the weather?

    Satellites monitor cloud movements and continuously measure temperature and humidity to predict how the weather will change today or tomorrow.


    B. Can AsomeBot tell forecast weather?

    Discuss with your friends and teachers how you can forecast the weather with AsomeBot.


    C. AsomeBot can download data from the Internet, such as weather, temperature, and humidity. Can we use this to tell the weather in the motion and sound of AsomeBot?


    D. When, where and how can we forecast the weather?



  • 3. learn which blocks to use

    • A. Connect to internet ([INTERNET] Group)

    Specify the name and password of the wireless router to which AsomeBot will be connected. The robot has to download data from the Internet, thus, the router must be connected to the Internet.

    import internet internet.connect("","")

    ① Enter the name of the Internet router.

    ② Enter the password of the Internet router.


    You can find the name of the wireless router you want to use in the smartphone WiFi list and connect to your smartphone.


    B. Open UDP socket ([INTERNET] Group)

    AsomeBot opens a port to communicate via the Internet.

    import udp_socket udp_socket.open(1234)

    ① The port number to be used. Use the number as is without modification.


    C. Get Weather data ([INTERNET] Group)

    Download data related to weather from the Internet.

    import internet internet.get_weather("Hanoi, VN")

    ① Enter the names of cities and countries to download weather data.


    Name value for each city

    Hanoi Vietnam: Hanoi, VN

    Seoul, S. Korea: Seoul, KR

    Los Angeles (the US): LA, us


    D. Measure distance ([SENSOR] Group)

    With an ultrasonic sensor, measure the distance of the obstacle in front. The value of the measured distance is stored in a variable name ‘Cm’. Sensors can report the weather when a person is standing or passing at a moderate distance.

    cm = hcsr04.get_distance()

    E. Repeat ([ADVANCE] Group)

    Repeat the commands in the block as many times as you want.

    for count in range(1): pass

    ① Specify the number of times to repeat.

    ② Add blocks to repeat.


    F. Repeat infinitely ([ADVANCE] Group)

    Run blocks inside blocks to run in order.

    while True: pass

    G. if ([ADVANCE] Group)

    Execute the block in the block when the detail input as condition is correct. If the condition is incorrect, the block following the if block will be executed.

    if i == :

    ① Input a variable to be used in the conditional statement. The default value is 'i' and changes to 'cm' when using an ultrasonic sensor.

    ② Select the default inequality to compare the conditions.

    ③ Input the default value to use in the condition. You need to add blocks to specify the conditional value.

    ④ Add blocks to be executed when the condition is correct.


    H. Data Block Character Value Block ([ADVANCE] Group)

    You can use one of the following two blocks for the conditional value of the conditional statement.

    1

    For the data block, enter and use the value you want to compare.

    ① Enter the data value to compare with the variable of the control statement.


    ''

    character value block should be used when using a character value for the conditional value or use a character value or input character received from the Internet as conditional value.

    ① Enter characters to be used as a condition.


    Character values related to weather

    Weather related character values you can download from the Internet are as follows:

    Rainy days: rain

    Sunny day: sunny

    Snowy day: snow


    Create a program that plays ‘Do’ ‘Re’ ‘Mi’ scales in order of 1 second if today’s weather value downloaded from the Internet is ‘sunny’.


    J. Attention ([BASIC] Group)

    AsomeBot may fall if connected to other motions when the feet have moved heavily. Putting [Attention] between each motion will let AsomeBot to stand at attention to prevent from falling.

    asomebot.home()

    K. Dance moves without Option ([DANCE] Group)

    asomebot.ballet()

    Perform ballet moves by standing the toes of AsomeBot.


    asomebot.wiggle()

    AsomeBot wiggles in place.


    asomebot.flap()

    AsomeBot flaps its feet.


    asomebot.warigari()

    AsomeBot performs Warigari with its feet.


    asomebot.yaho()

    AsomeBot lifts up the tiptoe of the feet. The opposite of ballet motion.


    asomebot.moonwalk()

    AsomeBot performs moonwalk dance.


    L. Dance Moves with Option ([DANCE] Group)

    Select the direction to move AsomeBot’s foot direction in the option of all dance moves.

    asomebot.mouse(1)

    Lift up one foot of AsomeBot and execute the motion.


    asomebot.swing(1)

    Lift up and fold back one foot of AsomeBot.


    asomebot.tick_tock(1)

    Fold one foot of AsomeBot back and wag once.


    M. Buzzer on ([SOUND] Group)

    Make the buzzer unable to use. To make sound with the buzzer, you must use this block. Note that if the block is not used first, you cannot use the block.

    turnoff_pins( (1, 5, 6, 7, 8) ) import music music.open(1)

    N. Buzzer off ([SOUND] Group)

    This allows you to use other parts again after using the buzzer. You cannot use other parts while using the buzzer. After playing the music, the block must be used to run other components.

    turnoff_pins( (1, 5, 6, 7, 8) ) import asomebot asomebot.ready(5, 6, 7, 8)

    O. Play for ([SOUND] Group)

    Using this block, you can make the sound you want using the frequency.

    music.note("C1",1*1000)

    ① Select the scale from the list that you want to play.

    ② Select the octave of scale from the list to play.

    ③ Specify the time to make the sound.


  • 4. Play Activity

    • A. Game Rule

    ① Let's talk about how and when to start weather forecasting.

    ② Let's talk about how AsomeBot can express the weather.

    ③ Besides the weather, talk about what other information (temperature, humidity, and fine dust) can be forecast.

    ④ Besides the water, talk about what other data services you can download and forecast from the Internet.


    B. Algorithm

    ① Set to download weather data from the Internet.

    ② Develop motions to run according to the weather.

    ③ Decide which conditions should be forecast when the condition is met and develop a program.

    ④ Develop a program that performs the motion for today's weather when the condition is met.

    ⑤ Add programs to make it more interesting.

  • 3. learn which blocks to use

    • A. Change to access point mode ([INTERNET] Group)

    Assign a name of the AsomeBot to be controlled by the smartphone.

    import internet internet.open_ap("")

    ① Enter a name you want to use. Only English and numbers can be used.


    ① Give AsomeBot a name. Only English and numbers can be used for naming AsomeBot. Once you assigned a name, the name will retain even when the power is turned on and off.

    ② check the name you specified and try connecting. first, find the name of asomebot you entered from the wifi list and connect to your smartphone. if a message pops up saying that the internet connection is unstable, do not press any buttons and wait for the message to disappear.


    B. Open UDP socket ([INTERNET] Group)

    AsomeBot opens a port to communicate via the Internet.

    import udp_socket udp_socket.open(1234)

    ① The port number to be used. Use the number as is without modification.


    C. Read Messages ([INTERNET] Group)

    Read messages sent from the smartphone.

    msg = udp_socket.read_text()

    G. Attention ([BASIC] Group)

    AsomeBot may fall if connected to other motions when the feet have moved heavily. Putting [Attention] between each motion will let AsomeBot to stand at attention to prevent from falling.

    asomebot.home()

    Create and run a program that poses stand at attention after ballet motions.


    H. Dance Moves without Option ([DANCE] Group)

    asomebot.ballet()

    Perform ballet moves by standing the toes of AsomeBot.


    asomebot.wiggle()

    AsomeBot wiggles in place.


    asomebot.flap()

    AsomeBot flaps its feet.


    asomebot.warigari()

    AsomeBot performs Warigari with its feet.


    asomebot.yaho()

    AsomeBot lifts up the tiptoe of the feet. The opposite of ballet motion.


    asomebot.moonwalk()

    AsomeBot performs moonwalk dance.


    Ballet is a motion that only stands at tiptoe. Therefore, combine with other motions and create more fun moves.


    I. Dance Moves with Option ([DANCE] Group)

    Select the direction to move AsomeBot’s foot direction in the option of all dance moves.

    asomebot.mouse(1)

    Lift up one foot of AsomeBot and execute the motion.


    asomebot.swing(1)

    Lift up and fold back one foot of AsomeBot.


    asomebot.tick_tock(1)

    Fold one foot of AsomeBot back and wag once.


    ① Mouse is made by Korean elementary school student. The student mimicked the scene whistling with moving feet in Mickey Mouse animation and named his project “Mouse”.

    ② Using Swing and TickTock motions, create a program that moves the feet at a time.


    J. Set angle of motor ([WALK] Group)

    The block controls 1 motor. Select one motor and specify the angle and speed of movement of the selected motor.

    asomebot.angles([0],[90-90],1*1000)

    ① Select the number of the motor to be moved.

    ② Enter the angle to move the selected motor.

    ③ Enter the time it takes for the motor to move.


    Using the AsomeBot motor number in the figure, create a program that lets AsomeBot's right foot (motor 3) angle to sustain at 30 degrees for 1 second and 90 degrees after 1 second.


    K. Set angles ([WALK] Group)

    The block controls angles of 4 motors at the same time. You can specify the angle you want of the 4 motors and rotate and set the moving time.

    asomebot.angles([0],[90-90],1*1000)

    ① Specify angles of 0, 1, 2 and 3 motors in order from the left.

    ② Enter the time it takes for the motor to move.


    Using the AsomeBot motor number in the figure, create a program that lets AsomeBot to lift up and down of the left and right feet alternatively.
    Hint. Left foot angle 150 degrees / Right foot angle 30 degrees

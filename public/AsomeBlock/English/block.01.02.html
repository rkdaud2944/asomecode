  • 1. Learning objectives

    • • You will learn the principle of the moving legs of the robot.
    • You can let AsomeBot walk forward or backward or to the side.
    • Learn about the rotation range of a 180-degree motor.



  • 2. Preview

    • A. Joints and muscles of robot

    Just like joints and muscles to move a person's body, the motor is each joint of the robot and power by which robot is moved.


    B. The rotation angle of servo motor

    • The server motor we will use in AsomeBot can only rotate up to 180 degrees.
    • AsomeBot uses four motors to walk and dance.
    • the 4 servo motors are all set at 90 degrees when AsomeBot stands at attention.


    C. Connecting the block-coding program

    To create a program that the AsomeBot moves by block-coding, you must connect the AsomeBot and PC (or smart device) with a USB cable as follows.


    ① Connect the small side of the USB cable to the port behind the AsomeBot.


    (Smart devices)

    ② Connect the other side of the USB cable to the OTG gender and then to the connection port on the smart device.


    (PC&Notebook)

    ③ Connect the other side of the USB cable to the USB port on the PC.


    ④ Click the [Connect] button at the top of the block coding screen. If a new window pops up and displays “Connected”, your PC (smart devices) is connected to AsomeBot.


    1. Closing the connection window terminates the connection with AsomeBot.

    2. If you disconnect and reconnect the USB cable to run AsomeBot, you need to click [Connect] for reconnection.



  • 3. Learn which blocks to use

    • A. Learning the block coding screen

    ① Connect: Connect with the learning tools.

    ② Execute: Execute the ready block.

    ③ Pause: Pauses the block from executing.

    ④ Save: Saves the ready block.

    ⑤ Load: Import the saved block file.

    ⑥ Learning tool selection tab: Select the desired learning tool. The block displays for each learning tool.

    ⑦ Block group tab: Blocks are distinguished and gathered by type.

    ⑧ Block: The command block to move learning tools.

    ⑨ Operation area: Drag and add the block to execute.

    ⑩ Python code: Display the added blocks in the Python language.

    ⑪ Code hide/show button: Show or hide the Python code display window.

    ⑫ Waste bin: You can delete a block by dragging and dropping it in the waste bin.


    B. Learn which blocks to use

    (1) Ready (BASIC Group)

    Before starting all programs, there must be a [Ready] block. By adding a [Ready] block in the work area, commands that prepare to move AsomeBot will execute automatically.

    import asomebot import hcsr04 hcsr04.open(3, 2) asomebot.ready(5, 6, 7, 8)

    • Prepare to use the ultrasonic sensor installed in AsomeBot
    • Prepare to use the four servo motors installed in AsomeBot


    (2) Attention (BASIC Group)

    • Make the AsomeBot to stand at attention.
    • At this time, the angle of all motors are set to 90 degrees.

    asomebot.home()

    • Connect AsomeBot and add the Ready and Attention blocks in order. Is the robot stand at attention?


    (3) Set angle of motor (WALK Group)

    You can select the motor for which you want to set the angle and set its angle and the time it takes for the motor to change its angle.

    asomebot.angles([0],[90-90],1*1000)

    ① select a number of motor that you want to set the angle of.

    ② Enter the angle at which the selected motor will set.

    ③ Enter the time it takes for the motor to set.

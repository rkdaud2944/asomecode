  • 1. Learning objectives

    • • Let AsomeBot walk forward, backward, left, and right.
    • Learn how to make the robot to wait for a specified second.



  • 2. Preview

    • A. How does AsomeBot walk?

    Think about how your feet move when you walk. Like a human being, AsomeBot walks in four different movements to take one step.

                                                        ① raise the left foot             ② Left foot forward

                                                        ③ raise the right foot             ④ Right foot forward


    B. Delay command

    By adding a Delay block following the Ready block, the robot waits for the specified time before the next block is executed To let the robot to walk in the desired gesture without a USB cable, create a delay time by adding a Delay block following the Ready block.

    import asomebot import hcsr04 hcsr04.open(3, 2) asomebot.ready(5, 6, 7, 8) delay(5) for i in range(5): asomebot.forward()



  • 3. Learn which blocks to use

    • A. Move forward (WALK Group)

    Walk forward by the specified number of steps as an option.

    for i in range(1): asomebot.forward()

    Input the number to steps to move forward after clicking. One step moves left and right foot at a time.


    • Create the block program that makes AsomeBot ready and to walk forward three steps. Check how far AsomeBot walked.


    B. Move backward (WALK Group)

    Walk backward by the specified number of steps as an option.

    for i in range(1): asomebot.backward()

    Input the number of steps to move backward after clicking. One step moves left and right foot at a time.


    • Create the block program that makes AsomeBot ready and to walk backward three steps. Check how far AsomeBot've walked.


    C. Turn Left (WALK Group)

    Turn left as many as specified as an option.

    for i in range(1): asomebot.turn_left()

    Input the number of steps to rotate to the left after clicking. To rotate 90 degrees, the robot should rotate 3 to 4 times.


    • Create the block program to make AsomeBot ready and to turn left 3 steps. Check the angle of how far the AsomeBot've turned to the left.


    D. Turn Right (WALK Group)

    Turn right as many as specified as an option.

    for i in range(1): asomebot.turn_right()

    Input the number of steps to rotate to the right after clicking. To rotate 90 degrees, the robot should rotate 3 to 4 times.


    • Create the block program to make AsomeBot ready and to turn right 3 steps. Check the angle of how far the AsomeBot've turned to the right.


    E. Delay (ADVANCE Group)

    Wait for the specified number of seconds to function.

    delay(1)

    Input the number of seconds to wait after a click.


    ① Create the block program that AsomeBot moves forward 5 steps, waits for 3 sec and walks backward 5 steps.


    ② When below block is run, what direction does AsomeBot look and stand?

    for i in range(3): asomebot.forward() for i in range(4): asomebot.turn_left() for i in range(5): asomebot.backward() for i in range(1): asomebot.turn_right() for i in range(2): asomebot.forward()


  • 4. Play activity

    • A. Game rules

    ① Make a program that moves 30 steps. (If the number of steps is small, increase the number of steps and set rules.)

    ② The robot can go anywhere on the outskirts of the bead.

    ③ A robot that steps on the most beads is the winner. (red beads, 10 points, yellow beads, 5 points)


    B. Algorithm

    ① The robot needs to step on a lot of beads within 30 steps. Set the direction and start position to step on a lot of red beads that give high points.

    ② Check how many steps are required to move a single column of beads.

    ③ Leave a delay of 5 seconds in the Delay block.

    ④ Make a program to move AsomeBot to the desired direction.


    C. Play game

    Play a game with your friends and see who gets to step on a lot of beads.

  • 2. Preview

    • A. How to transmit a sound

    Sound is that a vibration caused objects, instruments or people

    moves to air around us and transmits to our ear.


    B. How computers imitate sounds

    It can mimic the pitch height using the frequency of sound.

    It can imitate the strength and figure of sounds, when imitates sounds more precisely.


    3 elements of sound

    • strength : A intensity and weakness of sound
    • height : A high and low degree of sound
    • figure : It is also called tone

    To be distinguished by waveform

    The reason people have different voices


    C. Who made the scales?

    The seven-note scale, Do-Re-Mi-Fa-Sol-La-Si, is made by Pythagoras a famous mathematician.

    Pythagoras, who was passing by the blacksmith’s, heard people hammering iron with different hammers, got the formula for the sound, and made the seven-note scale that we use.


    D. Note and code

    If you use all the spelling of note, letters too much, so that program uses a code consisting of one character and octave.

    The scale we use most often is three octaves.



  • 3. Learn which blocks to use

    • A. Buzzer on (SOUND Group)

    This block makes buzzer available to use. This block must be used before the buzzer make a sound.

    CAUTION. Can't use the buzzer unless you use this block first.

    turnoff_pins( (1, 5, 6, 7, 8) ) import music music.open(1)

    B. Buzzer off (SOUND Group)

    It allows you to finish using the buzzer and use other parts again.

    You can’t use other parts during using the buzzer.

    After you finish to play, you must use this block to use other parts.

    turnoff_pins( (1, 5, 6, 7, 8) ) import asomebot asomebot.ready(5, 6, 7, 8)

    C. Make sound (SOUND Group)

    You can make sound you want by frequency with this block.

    music.tone(0) delay(1) music.mute()

    ① Enter the frequency that you want to sound

    ② Enter the playing-time


    • Input 196 to the frequency and listen to the sound. Which sound that corresponds to the scale can you hear?


    D. Play for (SOUND Group)

    You can make sound you want by frequency with this block.

    music.note("C1",1*1000)

    ① Select the note you want play.

    ② Select the octave.

    ③ Enter the playing-time


    • Play the Fa of 3-octaves for 0.5 sec.


    E. Beat and time

  • 4. Play activity

    • A. Game rules

    ① Using the rhyme notes, create a program that plays the nursery rhyme.

    ② Use the sound blocks you learned previously and create a program to play the music you want.


    B. Algorithm

    To play the music, you must first prepare to use a buzzer. And then, lay out the blocks according to each scale and beat, and at the end of the performance, you lay out blocks that turn off the buzzer.


    C. Play game

    Listen to the music and see whose music sounds better.

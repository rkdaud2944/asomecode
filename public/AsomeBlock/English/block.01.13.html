  • 3. Learn which blocks to use

    • A. Change to access point mode ([INTERNET] group)

    Name your AsomeBot controlled by smartphone.

    import internet internet.open_ap("")

    ① Give the AsomeBot a name that you want. Only English and numbers can be use in the name of the Asomebot.


    B. Open UDP socket ([INTERNET] group)

    Open a port for AsomeBot to communicate over the Internet.

    import udp_socket udp_socket.open(1234)

    ① Port number. Use it as it is without modification


    C. Read Messages ([INTERNET] group)

    Read messages from smartphone.

    msg = udp_socket.read_text()

    D. Repeat infinitely ([ADVANCE] group)

    [Repeat infinity] makes blocks that are bundled into it to continuously(endlessly) run in order.

    This time, This block is used by the smartphone app to wait for messages to be sent, and then execute commands that follow messages when prompted.

    while True: pass

    E. Repaeat ([ADVANCE] group)

    This block allows AsomeBot to repeat a specific movement

    or a bundle of movements as many times as you want.

    for count in range(1): pass

    ① Enter the number of times to repeat an movement

    ② Add blocks to repeat.


    F. if ([ADVANCE] group)

    [if] makes blocks that are bundled into it to run, when it corresponds with the condition. If it doesn't correspond with the condition, the blocks behind the [if] are run.

    if i == :

    ① Enter the variable that is used for the conditional statement. Basic value is ‘I’

    ② Select a inequality sign to compare with the conditions

    ③ Enter the reference value to use for the condition. At this time, you must add a conditional value block.

    ④ Add a block to run when conditions are met.

    ② Attention : This button send an H message for Asomebot to stand at attention.

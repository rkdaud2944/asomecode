    • A. Let the robot to dance like idols.

    You can use AsomeBot's WiFi function to create several robots dance at the same time.


    How to send signals via WiFi

    ① 1:1 operation

    Connect your smartphone and AsomeBot to operate one by one.

    ② Operate several robots at a time

    After the smartphone sends a signal to the router, the router transmits signals to several AsomeBots for operation.

  • 3. learn which blocks to use

    • A. Connect to internet ([INTERNET] Group)

    Specify the name and password of the wireless router to which AsomeBot will be connected. To use the block, you must have a wireless Internet router. However, the Internet router does not need to be connected to the Internet.

    import internet internet.connect("","")

    ① Enter the name of the Internet router.

    ② Enter the password of the Internet router.


    Find the Internet router on your smartphone. You can find the name of the wireless router you want to use in the WiFi list and connect to your smartphone. If a message pops up because the Internet connection is unstable, do not press any buttons and wait for the message to disappear.


    B. Open UDP socket ([INTERNET] Group)

    AsomeBot opens a port to communicate via the Internet.

    import udp_socket udp_socket.open(1234)

    ① The port number to be used. Use the number as is without modification.


    C. Read Messages ([INTERNET] Group)

    Read messages sent from the smartphone.

    msg = udp_socket.read_text()

    D. Repeat infinitely ([ADVANCE] Group)

    Run blocks inside blocks to run in order. Here, you will wait for your smartphone App to send a message, and execute the command according to the message.

    while True: pass

    E. Repeat ([ADVANCE] Group)

    Repeat the commands in the block as many times as you want.

    for count in range(1): pass

    ① Specify the number of times to repeat.

    ② Add blocks to repeat.


    F. if ([ADVANCE] Group)

    Execute the block in the block when the detail input as condition is correct. If the condition is incorrect, the block following the if block will be executed.

    if i == :

    ① Input a variable to be used in the conditional statement. The default value is ‘i’.

    ② Select the default inequality to compare the conditions.

    ③ Input the default value to use in the condition. You need to add blocks to specify the conditional value.

    ④ Add blocks to be executed when the condition is correct.

    H. Attention ([BASIC] Group)

    AsomeBot may fall if connected to other motions when the feet have moved heavily. Putting [Attention] between each motion will let AsomeBot to stand at attention to prevent from falling.

    asomebot.home()

    Create and run a program that poses stand at attention after ballet motions.


    I. Dance Moves without Option ([DANCE] Group)

    asomebot.ballet()

    Perform ballet moves by standing the toes of AsomeBot.


    asomebot.wiggle()

    AsomeBot wiggles in place.


    asomebot.flap()

    AsomeBot flaps its feet.


    asomebot.warigari()

    AsomeBot performs Warigari with its feet.


    asomebot.yaho()

    AsomeBot lifts up the tiptoe of the feet. The opposite of ballet motion.


    asomebot.moonwalk()

    AsomeBot performs moonwalk dance.


    Combine dance moves that don’t require an option and create a dance program.


    J. Dance Moves with Option ([DANCE] Group)

    Select the direction to move AsomeBot’s foot in the option of all dance moves.

    asomebot.mouse(1)

    Lift up one foot of AsomeBot and execute the motion.


    asomebot.swing(1)

    Lift up and fold back one foot of AsomeBot.


    asomebot.tick_tock(1)

    Fold one foot of AsomeBot back and wag once.

  • 4. Play Activity

    • A. Game Rule

    ① You need to decide what music to use for a dance competition and the part to be used (less than 30 seconds).

    ② Draw a great performance stage on paper with your friends.

    ③ Form a team of 3-5 AsomeBots.

    ④ Create a program so that AsomeBot of a team to dance to the music in harmony.


    B. Algorithm

    What should you do to create a program that lets multiple AsomeBots dance in the same motion?

    ① Determine how many seconds the beat of the measure of a song is used.

    ② Check the time it takes to perform each dance move.

    ③ Arrange the dance moves to the music and check where the beat of motion and music don’t fit.

    ④ Inset the [Delay] command in the area where the beat does not match the move.

    ⑤ Specify each AsomeBot's move differently where needed.

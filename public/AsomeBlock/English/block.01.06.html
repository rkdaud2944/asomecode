  • 3. Learn which blocks to use

    • A. Move forward (WALK Group)

    Walk forward by the specified number of steps as an option.

    for i in range(1): asomebot.forward()

    Input the number of steps to walk forward after clicking. At one step, the left and right foot move at a time.


    B. Move backward (WALK Group)

    Walk backward by the specified number of steps as an option.

    for i in range(1): asomebot.backward()

    Input the number of steps to walk backward after clicking. At one step, the left and right foot move at a time.


    C. Turn Left (WALK Group)

    Turn left as many as specified as an option.

    for i in range(1): asomebot.turn_left()

    Input the number of steps to rotate to the left after clicking. To turn 90 degrees, the robot has to turn 3 to 4 times.


    D. Turn Right (WALK Group)

    Turn right as many as specified as an option.

    for i in range(1): asomebot.turn_right()

    Input the number of steps to rotate to the right after clicking. To turn 90 degrees, the robot has to turn 3 to 4 times.


    E. Delay (ADVANCE Group)

    Wait for the specified number of seconds to function.

    delay(1)

    Input the time to wait after clicking in the unit of seconds.


    ① Try to let AsomeBot walk along the line as below.

    ② Assemble blocks so that AsomeBot moves the fastest way along the line.



  • 4. Play activity

    • A. Game rules

    ① Set the starting point of AsomeBot.

    ② Choose the buildings that AsomeBot should visit, and set up each object that can be knock down for each building.

    ③ The objects should be knock down to be recognized to have visited.

    ④ The person who visited all the appointed buildings in the least time using the program is the winner.

    ⑤ Always move along the road.


    B. Algorithm

    ① Set the path for AsomeBot to travel.

    ② Measure the distance of the road and check in how many steps the robot can visit the building.

    ③ Create a program according to the distance and direction the robot will travel.


    C. Play game

    Play a game and see who can come around by the specified building fast and complete the mission.

    AsomeBot opens a port to communicate via the Internet.

    import udp_socket udp_socket.open(1234)

    ① This is a port number to be used. Use the number as is without modification.

    To control AsomeBot wirelessly, install AsomeController App. With this App, you can send messages you want an control AsomeBot. Find ‘asomeit’ on App Store and install ‘Asome Controller’. Installs the App that allows you to control your AsomeBot wirelessly.


    I. Messages you can send with the Asome Controller

    You can execute various motions using the Asome Controller App.

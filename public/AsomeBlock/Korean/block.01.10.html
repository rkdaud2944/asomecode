  • 3. 사용 블록 알아보기

    • 가. 무한 반복 (ADVANCE 그룹)

    블록 안의 블록을 순서대로 계속 실행되도록 합니다.

    while True: pass

    나. 만약에(if) (ADVANCE 그룹)

    조건으로 입력한 내용이 맞을 때 블록 안의 블록을 실행시킵니다. 조건에 맞지 않는다면 [만약에(if)] 블록 뒤에 있는 블록이 실행됩니다.

    ① 조건문에 사용할 변수를 입력합니다. 기본 값은 ‘i’이며 센서 블록을 추가하면 해당되는 변수가 추가되어 선택할 수 있습니다.

    ② 조건을 비교할 기준 부등호를 선택합니다.

    ③ 조건에 사용할 기준 값을 입력합니다. 이때 조건 값을 지정할 블록을 추가해야 합니다.

    ④ 조건에 맞을 때 실행할 블록들을 추가합니다.


    다. 만약에, 아니라면(if else) (ADVANCE 그룹)

    조건으로 입력한 내용이 맞으면 [만약에(if)] 다음의 블록들이 실행되고 조건에 맞지 않으면 [아니라면(else)] 안의 블록들이 실행됩니다.

    ① 조건문에 사용할 변수를 입력합니다. 기본 값은 ‘i’이며 센서 블록을 추가하면 해당되는 변수가 추가되어 선택할 수 있습니다.

    ② 조건을 비교할 부등호를 선택합니다.

    ③ 조건에 사용할 기준 값을 입력합니다. 이때 조건 값을 지정할 블록을 추가해야 합니다.

    ④ 조건에 맞을 때 실행할 블록들을 추가합니다.

    ⑤ 조건에 맞지 않을 때 실행할 블록들을 추가합니다.


    라. 초음파 센서 거리측정 (SENSOR 그룹)

    초음파 센서를 이용해서 앞에 있는 장애물의 거리를 측정합니다. 측정된 거리 값은 ‘Cm’라는 이름의 변수에 저장됩니다.

    cm = hcsr04.get_distance()

    마. 부저 켜기 (SOUND 그룹)

    부저를 사용할 수 있는 상태로 만듭니다. 부저로 소리를 내기 전에는 반드시 이 블록을 사용해야 합니다. 이 블록을 먼저 사용하지 않으면 부저를 사용할 수 없으니 주의하세요.

    turnoff_pins( (1, 5, 6, 7, 8) ) import music music.open(1)

    바. 부저 끄기 (SOUND 그룹)

    부저의 사용을 마치고 다시 다른 부품들을 사용할 수 있게 해줍니다. 부저를 사용하는 동안에는 다른 부품을 사용할 수 없습니다.

    turnoff_pins( (1, 5, 6, 7, 8) ) import asomebot asomebot.ready(5, 6, 7, 8)

    사. 주파수로 소리내기 (SOUND 그룹)

    이 블록을 이용하면 주파수를 이용해서 원하는 소리를 낼 수 있습니다.

    music.tone(0) delay(1) music.mute()

    ① 내고 싶은 소리의 주파수를 입력합니다.

    ② 소리를 낼 시간을 지정합니다.


    아. 음계로 소리내기 (SOUND 그룹)

    이 블록을 이용하면 주파수를 이용해서 원하는 소리를 낼 수 있습니다.

    music.note("C1",1*1000)

    ① 연주하고 싶은 음계를 목록에서 선택합니다.

    ② 연주할 음계의 옥타브를 목록에서 선택합니다.

    ③ 소리를 낼 시간을 지정합니다.


    ① 초음파 센서로 측정한 거리에 따라 동작을 다르게 하는 프로그램을 만들어 보세요.
    • 5CM 이하일 때 뒤로 2걸음
    • 5CM 초과, 10CM 이하일 때 [플랩] 3번 후 차렷
    • 10CM 초과, 15CM 이하일 때 [문워크]
    ② 초음파 센서로 측정한 거리에 따라 동작을 다르게 하는 프로그램을 만들어 보세요.
    • 10CM 이하일 때 3옥타브 도레미파솔라시를 각 1초씩
    • 10CM 초과, 20CM 이하일 때 주파수 300부터 50씩 감소하면서 1초씩

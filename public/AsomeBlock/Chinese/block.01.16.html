    B. 创造新的舞蹈动作

    您可以用3种方法编制新的舞蹈动作。

    1. 将现有动作组合成一套动作。这样，可以将多个动作捆绑成一套动作并加以利用。

    2. 一次调整一个伺服电机的角度及运行时间。此时，要指定将运行的伺服电机。

    3. 一次调整多个伺服电机的角度及运行时间。此时，可以一次调整各个电机的角度及运行速度。如下所示操作，就能编制一个2秒钟内抬起两只脚并放下的动作。


    C. 调整运行时间

    如果电机运行时间分别调整为1秒和0.1秒，那么，即使是如下所示的同样动作也会因速度差异而显得完全不同。



  • 3. 了解要用的程序块

    • A. access point mode（接入点模式）（[INTERNET（互联网）]组）

    指定用智能手机控制的AsomeBot名称。

    import internet internet.open_ap("")

    ① 输入您要用的名称。只能使用英语和数字。


    ① 为AsomeBot设定一个名称。只有英文和数字才可用于AsomeBot的名称。设定名称后，即使打开或关闭电源，也依然保留其名称。

    ② 检查您设定的名称并尝试连接。先从WiFi列表中查找所输入的AsomeBot名称，并连接到您的智能手机。如果弹出一条信息显示互联网连接不稳定，请不要按任何按钮，要等待其信息消失。


    B. Open UDP socket（打开UDP套接字）（[INTERNET（互联网）]组）

    打开一个AsomeBot可通过互联网进行通信的端口。

    import udp_socket udp_socket.open(1234)

    ① 这是要用的端口号。不要做任何的改动，原封不动地使用端口号。


    C. Read Messages（读取信息）（[INTERNET（互联网）]组）。

    读取智能手机发送的信息。

    msg = udp_socket.read_text()

    D. Repeat infinitely（无限重复）（[ADVANCE（逻辑条件）]组）

    按顺序持续运行区块中的程序块。此时，等待智能手机应用程序发送的指令信息，并按照指令执行命令。

    while True: pass

    E. if（如果） ([ADVANCE（逻辑条件）] 组)

    所输入的条件正确时，运行区块中的程序块。如果条件不正确，就运行if区块之后的程序块。

    if i == :

    ① 输入用于条件语句的变量。默认值为“i”。

    ② 选择比较条件的标准不等式。

    ③ 输入用于条件的默认值。在此，需要您添加指定条件值。

    ④ 添加条件正确时要运行的程序块。


    F. Character Value（字符值）（[ADVANCE（逻辑条件）]组）

    这用于表示所输入的字符，而不是变量或数字。

    ''

    ① 输入要用作条件的字符。


    ① 假设您编制了如下的程序块，试着说明一下其如何运行：


    G. Attention（立正）（[BASIC（基础）]组）

    脚部动作幅度大的时候如果继续其他动作，AsomeBot就会跌倒。每次动作之间插入[Attention（立正）]使AsomeBot立正，就会防止跌倒。

    asomebot.home()

    编制一个芭蕾舞动作后立正的程序并加以运行。


    H. Dance Moves without Option（单个舞蹈动作）（[DANCE（舞蹈）]组）

    asomebot.ballet()

    AsomeBot用脚尖站着做Ballet（芭蕾舞）般的动作。


    asomebot.wiggle()

    AsomeBot在原地踏步。


    asomebot.flap()

    AsomeBot拍脚。


    asomebot.warigari()

    AsomeBot做出Warigari动作。（改为：双脚在地上来回蹭的动作）


    asomebot.yaho()

    AsomeBot抬起脚尖。是与[Ballet（芭蕾舞）]相反的动作。


    asomebot.moonwalk()

    AsomeBot走太空步。


    Ballet（芭蕾舞）只是一种用脚尖站立的动作。同其他动作结合起来编制一下更多有趣的动作。


    I. Dance Moves with Option（带选项的舞蹈动作）（[DANCE（舞蹈）]组）

    所有舞蹈动作选项均选择的是AsomeBot移动的脚部方向。

    asomebot.mouse(1)

    AsomeBot抬起一只脚做出动作。


    asomebot.swing(1)

    AsomeBot抬起一只脚向后弯曲。


    asomebot.tick_tock(1)

    Asomebot向后弯曲一只脚，并摇动一下。


    ① Mouse是由韩国小学生制作的动作。他模仿动画中米老鼠晃动着脚吹口哨的动作而编制，并命名为“Mouse”。

    ② 利用Swing和TickTock动作编制一下两只脚分别动一次的程序。


    J. Set angle of motor（设定电机角度）（[WALK（走步）]组）

    该区块控制1个电机。可以选择一个电机并调整所选电机的角度和运行速度。

    asomebot.angles([0],[90-90],1*1000)

    ① 选择要运行的电机编号。

    ② 输入所选电机的动作角度。

    ③ 输入电机运行所需的时间。


    参照图中的AsomeBot电机编号，编制一个AsomeBot的右脚（3号电机）角度1秒钟为30度，接着1秒钟为90度的程序。


    K. Set angles（设定角度）（[WALK（走步）]组）

    该区块可同时控制4个电机的角度。您可以调整4个电机当中所需的电机角度与运行时间。

    asomebot.angles([0],[90-90],1*1000)

    ① 从左边起依次指定0、1、2、3号电机的角度。

    ② 输入电机运行所需的时间。


    参照图中的AsomeBot电机编号，编制一个AsomeBot交替抬起或放下左右脚的程序。
    提醒：左脚角度150度/右脚角度30度。
